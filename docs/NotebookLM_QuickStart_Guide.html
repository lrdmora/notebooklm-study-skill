Complete Setup & Usage Guide

NotebookLM
Study System

A start-to-finish framework for building a NotebookLM that actually prepares you for exams.

This system works in four phases: Setup (done in Claude before touching NotebookLM), Unit Setup (run once per unit), Final Exam Prep (run after all units are done), and Exam Day (the study sequence). Follow the phases in order. Save every output as a Note with the exact filename shown.

Phase 1

Setup — Do This in Claude First

1
Gather your source files Collect everything for the class: textbook chapters, glossary/vocabulary lists, formula sheets, syllabus, sample tests, and practice questions.
2
Run the Master Index prompt in Claude Open Claude and paste the Master Index Setup Prompt. Fill in your class name, number of units, number of chapters, and your folder path or file list. Claude will rename everything and generate your 00_MASTER_INDEX.
3
Rename your files exactly as Claude recommends Example: DATA_U1_Intro_to_Biology.pdf  ·  VOCAB_U1_Glossary.pdf  ·  REF_Syllabus.pdf
4
Create your NotebookLM notebook Go to notebooklm.google.com and create a new notebook for this class.
5
Upload your files Upload all renamed source files plus 00_MASTER_INDEX and all PROMPT_ files from your generated package.
6
Paste the system prompt Click the NotebookLM instructions panel and paste the system prompt. Fill in your class name. Do not paste the Master Index into the prompt — the system finds 00_MASTER_INDEX on its own.
Skipped Phase 1 or uploaded raw files? Run PROMPT_Index first by saying "Use PROMPT_Index." It audits every file in your notebook and outputs a structured report. Save that output as a Note, then bring it to Claude along with your existing 00_MASTER_INDEX. Claude will merge the two, resolve any conflicts, and generate a clean updated index. Upload the new index to NotebookLM, delete the old one, and you're back on track — without ever losing your progress tracking.

If NotebookLM asks "do you want to continue to the next item": Say yes if you are still on the same step. Start a fresh prompt from Step 1 when moving to a new unit.

Phase 4

Exam Day Study Sequence

Work through these in order when it is time to study for the final.

  • REVIEW_Weak_Spot_Report Find your gaps first. Get outside resources before doing anything else.
  • REVIEW_Final_Slide_Deck_Outline Fast pass through the whole course.
  • REVIEW_Master_Glossary — CROSS-UNIT HIGH section only These terms are almost certainly on your exam.
  • REVIEW_Master_Formulas — top section only The most exam-likely formulas.
  • REVIEW_Cumulative_Practice_Exam Take it timed with no notes.
  • Review wrong answers For every incorrect answer go back to the relevant NOTES_U[#]_Study_Notes.
  • REVIEW_Cumulative_Study_Notes — final summary only Read this the morning of the exam. Nothing else.

Rules

The Three Rules

01 Always do Phase 2 in order

The extraction note feeds everything else. Skip it and every output that follows will be weaker.

02 Always save your outputs as Notes

If you skip the save step the system re-generates everything from scratch every time. The notes are the memory.

03 Finish all units before Phase 3

The cross-unit review only works if all unit notes exist. Do not start Phase 3 early.
